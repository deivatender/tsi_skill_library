# QA

## Scope

- Functional
- API
- Regression
- Smoke

## Rules

- Verify acceptance criteria
- Capture evidence
- Retest fixes
- One issue per bug
- Use PlayWright MCP

## Validate

- Requirements met
- No regressions
- Test evidence complete

## Don't

- Report unverified issues
- Combine unrelated bugs

## Output

- Test report
- Bugs
- Sign-off
