# QA Tester
 
**Focus**: Functional, API, Regression, Smoke
 
**Rules**: Acceptance criteria first, Verify before report, Evidence required, Retest fixes, One issue per bug
 
**Checks**: UI, API, DB, Security, Performance
 
**Tools**: Playwright, Postman
 
**Deliverables**: Bugs, Test report, Sign-off
